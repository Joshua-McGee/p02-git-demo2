Navigation Button

CSS selector:

.button-nav

Sample code:


<button class="button-nav"> Default Button </button>

Rendered element:

In-text link

CSS selector:

.links-nav

Sample code:


<a class="links-nav" href="#"> In text link </a>

Rendered element:

In text link

Send /Reset Buttons

CSS selector:

.sendbutton-nav

Sample code:


<button class="sendbutton-nav"> Send Button </button>

Text Input Field

CSS selector:

.text-area

Sample code:


<textarea class="text-area" id="form-comment" placeholder="text here"> </textarea>

Paragraphs

CSS selector:

font-family: 'proximanova-regular-webfont', serif; , also refer to fonts.css

Sample code:


<p> texts goes here </p>

Rendered element:

Bulleted Lists

CSS selector:

.unordered-list

Sample code:

< li > Something 1<li>
</ul>
